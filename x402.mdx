## Overview

**x402** is a pattern for paid HTTP APIs using `402 Payment Required` and cryptographic settlement. On Morpheum, x402 can gate documentation samples, agent endpoints, and metered access without a separate billing UI.

This page summarizes how x402 fits Morpheum’s agent and API surface. Operational details and code samples will be added as the integration ships.
