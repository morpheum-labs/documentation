## Overview

Use the **Model Context Protocol (MCP)** to connect Morpheum-capable tools and agents to editors, IDEs, and automation that speak MCP.

This section documents how Morpheum exposes or consumes MCP servers for trading, risk, and documentation workflows. Details will expand as the integration stabilizes.
